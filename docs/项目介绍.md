平台：Windows。

功能：

- UI 界面操作
- 集成YoloV8
- 默认带基础精简模型。
- 用户可以使用本工具标注数据集
- 用户标注好数据集后，选择标注好的文件夹，可以自己训练模型。
- 也可以导入模型。
- 兼容多种Nvidia GPU。支持用户选择GPU型号。
- 不考虑CPU模式驱动Yolo。
- 用户可设置鼠标移动速度。分为：慢速， 正常， 快速，极快。
- 用户可设置是否模拟人类操作，包括：

  - 鼠标移动：通过开关控制是否加入曲线模拟人类。是否在增加移动随机时间。可以设置移动目标的抖动范围。
  - 鼠标移动成功率：控制鼠标移动的成功率。
  - 鼠标点击：通过开关控制是否加入模拟人类点击间隔，点击多次，点击时轻微偏移。
  - 鼠标点击失误率：鼠标点击的成功率。
  - 其他可能的模拟操作。
- 用户可以选择要识别的窗口，或者整个屏幕。支持多块屏幕时选择屏幕。
- 默认设置快捷键。支持修改快捷键。快捷键通过长按控制开启识别后鼠标移动。
- 可以配置是否移动到目标后自动触发鼠标点击，以及如何点击。这个配置可以让用户自己点击鼠标。
- 可以配置是否加入识别框，便于用户发现目标。
- 程序加密，防止破解。需要严格加密，对抗高难度破解。
26→- **技术方案**：
27→
28→  - **开发语言**：Python 3.11.9 (锁定 3.11 系列的成熟版本)
29→  - **核心框架**：YOLOv8 + PyTorch (CUDA 加速，原生支持 PT 模型热加载)
30→  - **采集技术**：Desktop Duplication API (DDA) + MSS 兼容模式
31→  - **行为模拟**：Win32 API (SendInput) 步进插值与随机噪声模拟
32→  - **UI 界面**：PySide6 (Qt for Python)
33→  - **程序形态**：完全自包含 (Self-contained)，集成 Python 运行时与 CUDA 环境
- 程序完全自包含：用户下载即可运行，无需安装 Python、CUDA、驱动或其他任何第三方工具/依赖。
- 程序需要输入密钥来开启。密钥控制程序的可用时长。

  - 最佳情况是无需联网，也可以通过密钥控制市场。并且密钥无法复用。
  - 如果密钥功能必须联网，则需要支持与服务端同步密钥信息。程序运行期间每小时发送执行时间。
  - 如果有其他更好的机制，可以讨论。
