## 8. 常见问题 (FAQ)

- **无法加载 Syscall 驱动**：由于 `Syscall Input` 依赖于 C 扩展，如果您是直接运行源码且没有编译该扩展，系统会自动回退到 `Win32 Input` 模式。
- **显卡驱动初始化失败**：请确保您的显卡支持 CUDA 12.8，并已安装最新的 NVIDIA 驱动。
- **缺失模型文件**：程序默认加载 `base.pt`。如果缺失，请在 `configs/config.json` 中更正模型路径。
