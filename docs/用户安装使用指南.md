# AutoX 安装与使用指南

欢迎使用 AutoX - 基于 AI 的自动化控制系统。为了实现极致的性能和极小的体积，本项目采用了 **“核心编译 + 外部依赖”** 的分发方案。

---

## 一、 环境准备 (仅需一次)

在运行 AutoX 之前，请确保你的电脑满足以下基础要求：

### 1. 安装 NVIDIA 显卡驱动
*   确保你的显卡驱动版本较新（建议 570.xx 以上），以支持 CUDA 12.8。
*   你可以通过 GeForce Experience 或 [NVIDIA 官网](https://www.nvidia.com/Download/index.aspx) 更新。

> **注意**: 你不需要手动安装 Python 或 Visual C++ 运行库。AutoX 已内置了这些环境的自动安装程序。

---

## 二、 部署步骤

当你收到 AutoX 的发布包后，请按照以下步骤操作：

### 1. 一键初始化环境
双击运行根目录下的 **`init_env.bat`**。
*   该脚本会自动安装 **Visual C++ 运行库**（如果你的电脑没有的话）。
*   该脚本会自动下载并配置内置的 **Python 运行时环境** (`python_runtime`)。
*   它会自动下载并安装 Torch (约 2GB)、PySide6、OpenCV 等重型依赖。
*   **注意**: 这一步取决于你的网络速度，通常需要 5-10 分钟。如果中途报错，请检查网络并重新运行。

### 2. 准备模型
*   确保 `models/` 文件夹下存有你的训练模型（如 `best.engine` 或 `yolo11n.pt`）。
*   推荐使用 `.engine` 格式以获得最高推理性能。

---

## 三、 运行程序

### 1. 启动 AutoX
*   双击根目录下的 **`AutoX.exe`**。
*   程序启动时会自动加载 `python_runtime` 环境，如果检测不到会弹出错误提示。

### 2. 基础配置
*   在 GUI 界面中选择你的模型路径。
*   选择输入驱动（推荐使用 `Syscall Input`）。
*   设置你的热键和自瞄范围。

---

## 四、 常见问题 (FAQ)

#### Q1: 启动时提示 "WinError 1114: 动态链接库(DLL)初始化例程失败"
*   **原因**: 笔记本电脑在省电模式下限制了显卡调用。
*   **解决**: 
    1. 接上电源线。
    2. 在 `Windows 设置 -> 系统 -> 屏幕 -> 图形` 中，点击 `浏览` 找到 `AutoX.exe`，将其设置为 **“高性能”**。

#### Q2: 运行 `init_env.bat` 速度太慢
*   **原因**: 默认从官方服务器下载。
*   **解决**: 脚本已配置镜像源，但如果依然慢，请尝试开启/关闭代理，或在命令提示符中手动运行 `pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple`。

#### Q3: 杀毒软件报毒
*   **原因**: Nuitka 编译的二进制文件经常会被误报（尤其是涉及底层驱动调用的 Syscall 部分）。
*   **解决**: 请将 AutoX 所在目录加入杀毒软件的白名单。

---

## 五、 开发者说明 (如何重新打包)

如果你修改了源码并想重新生成 `AutoX.exe`：
1.  安装 Visual Studio 2022 (勾选 "C++ 桌面开发")。
2.  在当前环境下运行 `python build.py`。
3.  生成的单文件 EXE 位于 `dist/` 目录下。
